# 2D Ray Tracing Simulator

## Описание
Интерактивный симулятор трассировки лучей в 2D-пространстве с:
- Режимом зеркальных отражений
- Визуализацией BVH-оптимизации
- Генерацией случайных сцен
- Настройкой параметров в реальном времени

## Требования
- Python 3.8+
- Библиотеки: PyQt6

## Установка
1. Установите зависимости:
   pip install PyQt6
2. Запустите программу:
   python Ray_Tracing_2D.py


## Настройки
◉ Длина луча (по умолчанию: 300)
- Определяет максимальную дистанцию, на которую распространяются лучи

◉ Угол между лучами (по умолчанию: 2°)
- Шаг угла для генерации лучей (1-360°)
- Меньшие значения = больше лучей = плавнее освещение

◉ Количество стен (по умолчанию: 9)
- Число случайно генерируемых препятствий

◉ Длина стен (по умолчанию: 300)
- Определяет размер каждого препятствия

### Режимы работы:
◉ Оптимизация BVH (выкл. по умолчанию)
- Включает иерархическую структуру для ускорения расчетов

◉ BVH контур (только с оптимизацией)
- Визуализирует зоны проверки пересечений

◉ Режим зеркал (выкл. по умолчанию)
- Включает многоуровневые отражения
- Доп. параметры при включении:
  ▪ Макс. отражений (по умолчанию: 3)
  ▪ Потеря прозрачности (по умолчанию: 30%)

◉ Счетчик проверок
- Показывает количество вычисленных пересечений
- Индикатор производительности
  

## Управление
- Любая кнопка мыши: перемещение источника света
